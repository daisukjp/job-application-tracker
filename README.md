# Job Application Tracker

## Overview

A minimal job application tracker built with Next.js, Supabase, and TanStack Query.  
It supports application management, analytics, and follow‑up reminders.

## Features

- Applications list with search, filtering, sorting, and optimistic updates
- Create/edit with validation
- Analytics dashboard with charts
- Follow‑up reminders (overdue / today / upcoming)
- Supabase-backed persistence

## Tech Stack

- Next.js (App Router)
- TypeScript
- Supabase (Postgres)
- TanStack Query
- Tailwind CSS
- Recharts
- Vitest + Testing Library
